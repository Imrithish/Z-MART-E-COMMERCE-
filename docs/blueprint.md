# **App Name**: Z-MART Merchant

## Core Features:

- Secure Admin Login: Robust and secure authentication system for business owners to register and log in to the management panel.
- Intuitive Product Management: An interface allowing owners to easily add new products, edit existing listings (including details, images, and inventory), and organize their catalog.
- Order Management Dashboard: A comprehensive dashboard for business owners to view, track, and manage all customer orders, including order status updates.
- Integrated Public Storefront: A responsive, customer-facing online store for browsing products, viewing details, and making purchases across all devices.
- Shopping Cart & Checkout: Core e-commerce functionality enabling customers to add products to a shopping cart and proceed through a streamlined checkout process.
- AI Product Description Generator: A generative AI tool that assists business owners in creating compelling and descriptive product descriptions based on key features or inputs.

## Style Guidelines:

- Primary color: A deep, professional blue (#173B82), evoking trust and reliability.
- Background color: A very light blue-gray (#F0F2F4), providing a clean and expansive canvas.
- Accent color: A subtle, cool light gray (#B9BDC6), chosen to highlight interactive elements and provide visual separation while maintaining a professional feel.
- Body and headline font: 'Inter' (sans-serif) for its modern, clean, and highly legible appearance across all screen sizes, contributing to a professional aesthetic.
- Use clear, universally recognizable icons with a minimalist, outline-based style to maintain a professional and uncluttered user interface.
- Responsive grid-based layouts ensure optimal display and usability across desktop, tablet, and mobile devices for both the merchant panel and public storefront.
- Subtle, non-distracting transition animations for UI elements and page changes, enhancing user experience without appearing flashy.
